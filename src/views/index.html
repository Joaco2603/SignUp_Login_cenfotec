El mejor sistema de login y creación de cuentas.

Que es este proyecto??

Este es un proyecto realizado con node js, mongodb, express y express-session, es un ejemplo de implementación de login y creación de cuentas con fines puramente educativos.

Visión, Misión y Valores

Visión

Lograr que los usuarios logren hacer login y sigup de manera eficiente.

Misión

Nuestra misión es desarrollar un login y signup.

Valores

Simplicidad, Confiabilidad, Innovación, Empatía.
